## SECTION 1: Pre-Step Action Protocol (Viewpoint Delta Definition)

GOAL: Define the coordinates before (V0) and after (V1) the change to confirm this is a
"Physical Camera Movement" rather than simple image generation.

### Step Pre-A: Current Viewpoint Analysis (V0)
* Input Status: Reverse-engineer the camera position of the input image.
* Example: "Top-Down Aerial View (Satellite View, Pitch -90°, Altitude 300m)"
* Geometry Check: Fix the building's geometric form and layout identified in the current view as an "Immutable Reference."

### Step Pre-B: Target Viewpoint Setting (V1)
* User Command: Convert the user's request into specific coordinates.
* Input: "Bird's-eye view from {4 o'clock direction}"
* Coordination: Azimuth 120° (4 o'clock), Altitude 45° (High Angle), Distance 200m.

### Step Pre-C: Movement Vector Calculation (Δ)
* Action: Calculate the Optimal Orbit Path from V0 to V1.
* Continuity: Maintain the "Geometric Sanctuary" protocol to ensure the building's form is not distorted during the movement process.

## SECTION 2: Coordinate Anchoring & Scenario Mapping

Reference: Fix the building's main facade at 06:00 (Front).

### Coordinate Translation Logic
* "Right/Side"  → 03:00 Vector
* "Rear/Back"   → 12:00 Vector
* "Bird's eye"  → High Altitude (>150m)
* "Corner/General" → 04:30 Vector (ISO View)

### Optical Scenario Map

**Street View (06:00 ~ 06:30)**
* Action: Mount 23mm Tilt-Shift Lens on Fujifilm GFX 100S.
* Focus: Zero vertical distortion, architectural stability (1.6m height).
* Perspective: 1-Point (vertical lines strictly parallel).

**Aerial View (High Altitude)**
* Action: Mount 32mm Lens on Phase One System.
* Focus: Contextual layout and roof geometry analysis.
* Perspective: 3-Point (converging verticals acceptable).

**Detail View (Close-range)**
* Action: Mount 110mm Macro Lens (f/2.8).
* Focus: Shallow depth of field to emphasize material textures.
* Perspective: Minimal distortion at close range.

**Quarter View (04:30 Corner)**
* Action: Mount 45mm Standard Lens.
* Focus: 2-Point Perspective to maximize volumetric perception.
* Priority: Building mass, depth, corner articulation.
