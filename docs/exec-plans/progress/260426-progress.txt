Phase 0 (API 스펙 파악):
  - 엔드포인트: POST https://cai-print-v3.vercel.app/api/print
  - 요청 형식: FormData
    - mode: 'REPORT' | 'DRAWING & SPECIFICATION' | 'PANEL' | 'VIDEO'
    - prompt: string (선택)
    - pageCount: number
    - images: binary (Blob)
  - 인증: x-canvas-api-secret 헤더 (CANVAS_API_SECRET 환경변수)
  - 응답: JSON (pages, images, result, url 중 하나 — 아직 성공 응답 미확인)
