### 검증 과정 (중요)

1. 초기 가설 "TMS y좌표 반전 필요" → **기각됨**
   - curl로 실측: 현재 XYZ y좌표(203128) → HTTP 200 정상
   - TMS 반전 y좌표(321159) → HTTP 404 존재하지 않음
   - 결론: VWorld xdworld.vworld.kr는 XYZ 좌표계 사용

2. 진짜 원인 3가지 확인:
   - **좌표 정규화 부재**: viewBox 원점이 1억+(114,468,428) → SVG float32 정밀도 한계
   - **타일 수량 폭발**: padding 2.5배 + zoom 19 = 196장 동시 로딩 → 브라우저 연결 풀 소진
   - **onError 영구 숨김**: 1장이라도 실패 시 display:none → 복구 불가

3. passive event listener: React onWheel은 passive 기본 → preventDefault() 불가
