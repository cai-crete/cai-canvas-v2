# 지적도 NodeCard [확장] 버튼 → ExpandedView 라우팅 수정

## 무엇을 했는가

### 문제
- `cadastral`, `map3d` 노드는 `artboardType: 'image'`로 생성됨
- NodeCard [확장] 버튼 클릭 → `expandedViewMode: 'default'` 설정
- ExpandedView에서 `artboardType === 'image' && expandedViewMode === 'default'` 조건(구 line 269)이
  `node.type === 'cadastral'` 조건(구 line 327)보다 먼저 매치 → SketchToImageExpandedView로 잘못 진입
- `map3d`도 동일한 문제

### 수정: components/ExpandedView.tsx
- `cadastral` 분기와 `map3d` 분기를 `artboardType=image` 조건 블록 **앞**으로 이동
- 새 라우팅 순서:
  1. elevation
  2. isSketchImageMode
  3. isSketchPlanMode
  4. planners
  5. print
  6. **cadastral** ← 이동됨
  7. **map3d** ← 이동됨
  8. artboardType=image + plan
  9. artboardType=image + image/default
  10. artboardType=image + image/plan (생성 결과)
  11. 기본 레이아웃

## 완료 범위
