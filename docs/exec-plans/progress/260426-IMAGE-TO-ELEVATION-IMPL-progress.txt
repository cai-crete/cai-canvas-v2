### 노드 흐름 (완성된 동작)
1. 이미지 아트보드 선택 → ELEVATION 탭 클릭
2. elevation 입력 노드 생성 (원본 이미지를 thumbnailData로 복사) + 즉시 ExpandedView 진입
3. ExpandedView: 원본 이미지 표시 + PROMPT 입력 + GENERATE 버튼
4. GENERATE 클릭 → 즉시 캔버스 복귀 + "ELEVATION GENERATING" toast
5. 백그라운드: Protocol A (Gemini Vision → AEPLSchema) → Protocol B (4방향 병렬 생성, front = passthrough)
6. 완료 시 handleGenerateElevationComplete → CrossGrid 합성 썸네일 노드 B 생성
7. 노드 B expand → CrossGrid 5-view + EXPORT 버튼 (ELEVATION SPEC 패널 포함)

### 주요 설계 결정
