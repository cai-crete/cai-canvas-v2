### 수정 내용

**src/lib/mcpClient.ts** — `convertProperty()` 함수의 array items 누락 버그 수정

```typescript
// 수정 전 (버그):
if (prop.items) result.items = convertProperty(prop.items);
// → prop.type === 'array' 이지만 items가 undefined이면 조건 false
// → items 필드 누락 → Gemini 400 INVALID_ARGUMENT 발생

// 수정 후:
if (prop.type === 'array') result.items = convertProperty(prop.items || { type: 'string' });
// → 배열 타입이면 항상 items 포함 (기본값: { type: 'string' })
```

**증상**: Gemini가 `tools[0].function_declarations[N].parameters.properties[X].items: missing field`
오류를 반환하여 MCP 도구 중 배열 파라미터를 가진 도구가 등록 자체에 실패했음.
