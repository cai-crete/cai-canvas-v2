# ELEVATION / PRINT 이미지 미선택 시 토스트 처리 완료

## 무엇을 했는가
`project_canvas/app/page.tsx` → `handleNodeTabSelect` 함수 2곳 수정

### 변경 1: ELEVATION - 노드 미선택 시 토스트 (아트보드 미선택 섹션)
- 기존: `if (type === 'viewpoint')` → 토스트
- 변경: `if (type === 'viewpoint' || type === 'elevation')` → 토스트
- 효과: 노드 미선택 상태에서 ELEVATION 클릭 시 사이드바 패널 열리지 않고 토스트만 표시

### 변경 2: PRINT - 이미지/스케치 노드 없을 때 토스트
- `if (type === 'print')` 블록 내 imageNodes.length === 0 케이스에
  `showToast('이미지를 선택해 주세요'); return;` 추가
- 효과: 이미지/스케치 아트보드 노드가 선택되지 않은 상태에서 PRINT 클릭 시
  사이드바 미열림 + 토스트만 표시 (비이미지 노드 선택된 경우도 포함)

## 완료 범위
- ELEVATION: 노드 미선택 → 토스트 ✓ / 비이미지 노드 선택 → 토스트 ✓ (기존) / 이미지 선택 → 정상 ✓
- PRINT: 이미지/스케치 미선택 → 토스트 ✓ / 이미지/스케치 선택 → print 노드 생성 + expand ✓
