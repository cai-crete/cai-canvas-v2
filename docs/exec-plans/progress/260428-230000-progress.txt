### 구현 요약 (commit ac26c70)

**핵심 흐름:**
1. `userInput` 텍스트에서 용도지역 + 주소 감지
2. `extractSido()` → 서울특별시, 경기도 등 광역자치단체 추출
3. `mcpSearchOrdinance("{sido} 도시계획 조례")` → 자치법규 검색
4. 정규식으로 "{sido} 도시계획 조례" 타이틀 정확 매칭 → ordinSeq 추출
   - 버그: 처음엔 `지자체: {sido}` 만 매칭 → "서울특별시 노후계획도시 정비에 관한 조례"를 먼저 선택
   - 수정: `[^\n]*도시계획 조례\s*\n\s*지자체:\s*{sido}\s*\n` 으로 타이틀도 확인
5. `mcpGetOrdinance(ordinSeq)` → TOC 조회
6. `findArticleNumberInToc(toc, "용도지역 안에서의 건폐율")` → 제44조 동적 추정
7. `findArticleNumberInToc(toc, "용도지역 안에서의 용적률")` → 제48조 동적 추정
   - TOC 목차 항목 순서 = 조문 번호 (1-indexed)
8. 병렬로 제44조, 제48조 조문 조회 → Gemini lawsText에 주입

**서울특별시 도시계획 조례:**
- MST: 2116657 (20260330 시행, 동적 검색으로 확보)
