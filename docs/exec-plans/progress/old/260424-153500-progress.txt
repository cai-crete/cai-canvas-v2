### 핵심 아키텍처 결정
- Hybrid Rendering: canvas는 드로잉 스트로크만, 이미지는 DOM <img> (z=2)
- cursor tool 클릭 시 이미지 hit-test → imageEditingActive 토글
- resize/rotate 핸들이 canvas에 포인터 캡처 → handlePointerMove가 transform 처리
- imageTransform은 월드 좌표 (zoom=1 기준), 화면 변환 시 *zs + offset 적용
