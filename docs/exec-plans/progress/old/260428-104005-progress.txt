작업: Canvas ↔ Print 노드 연결 — 브라우저 테스트 3대 버그 수정

완료된 작업:
1. Phase 1~8 서버 테스트 완료 (이전 파일 참조)
2. 브라우저 테스트 후 발견된 3가지 버그 수정:

[Fix 1] Bebas Neue 폰트 미적용
  - 원인: @cai-crete/print-components 패키지가 text-ui-title / text-ui-subtitle / text-ui-caption 클래스 사용하는데 Canvas 앱의 globals.css에 미정의
  - 수정: globals.css에 세 클래스 추가 (--font-family-bebas / --font-family-pretendard 변수 사용)
  - 파일: project_canvas/app/globals.css

[Fix 2] 이미지 삽입 시 썸네일 슬롯 깨짐
  - 원인: React 19 Strict Mode 이중 effect 실행 → useMemo가 캐시한 Blob URL을 cleanup이 revoke해버려 src 무효화
  - 수정: next.config.ts에 reactStrictMode: false 추가
  - 파일: project_canvas/next.config.ts

[Fix 3] GENERATE 버튼 클릭 시 "Failed to fetch" 오류
  - 원인: BFF에서 req.body ReadableStream + duplex:'half' 스트리밍 방식이 Next.js 15 dev 환경에서 불안정
  - 수정: req.arrayBuffer()로 본문 전체 버퍼링 후 ArrayBuffer로 upstream 전송. content-length도 실제 버퍼 크기로 정확하게 설정
  - 파일: project_canvas/app/api/print-proxy/api/print/route.ts

아직 완료되지 않은 작업:
