## 원인 분석
- PRINT vs IMAGE 패턴 비교를 통해 근본 원인 특정
- 기존 코드: `if (type === 'image' && selectedNodeIds.length >= 2)` — 외부 조건이 너무 좁음
  - Ctrl+클릭 다중선택 미구현 → 러버밴드 드래그만 다중선택 가능
  - 단일 카드 클릭 시 `selectedNodeIds = [id]` → `length >= 2` 불충족 → 낙하
  - 낙하 시 `createAndExpandNode('image')` 실행 → sketchInputImages 없는 노드 생성
- PRINT는 `if (type === 'print')` 후 바로 `imageNodes.length > 0` 체크 → 단일 선택도 동작
