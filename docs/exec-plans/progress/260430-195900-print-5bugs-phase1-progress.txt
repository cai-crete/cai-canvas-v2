## Phase 1 수정 상세 내역

### 수정 파일 1: Report_template.html
- 경로: project.10_print/sources/document_template/Report_template.html
- 목적: AI가 <script> 블록을 텍스트로 복사하여 이중 페이지를 생성하는 근본 원인 제거

#### 변경 1: 목차 영역 정적 변환 (구 98~105행)
- 변경 전: <script>for(let i=1; i<=6; i++) document.write(...) — JS 루프로 동적 생성
- 변경 후: 정적 HTML 6개 항목 (01~06)
- 사유: AI가 script 내부 template literal을 인식 못함 → R1 버그 원인

#### 변경 2: 내지 영역 정적 변환 (구 109~128행)
- 변경 전: <script>[...].forEach(p => document.write(...)) — 2개 내지를 JS로 동적 생성
- 변경 후: 정적 <div class="page"> 2개 (레이아웃 A: 이미지 1장, B: 이미지 2장)
- 사유: AI가 script를 텍스트로 복사 + 별도 정적 HTML도 생성 → R2 더미 페이지 직접 원인

#### 변경 3: 내지 레이아웃 D 그리드 정적 변환 (구 171~175행)
- 변경 전: <script>for(let i=1; i<=4; i++) document.write(...) — JS 루프
- 변경 후: 정적 HTML 4개 항목

#### 변경 4: 하단 브라우저 인터랙션 script 제거 (구 179~207행)
- 변경 전: <input type="file">, 이미지 업로드 로직, 텍스트 리밋 로직
- 변경 후: 완전 제거
- 사유: AI 프롬프트 전용 — 브라우저 JS 불필요

#### 변경 5: onclick="u(this)" 속성 전수 제거

#### 최종 템플릿 페이지 구성 (정적 HTML):
1. 표지 (Page 1) — 기존 유지
2. 목차 (Page 2) — script→정적 6항목
3. 내지 A (Page 3) — 이미지 1장 + 설명 (신규)
4. 내지 B (Page 4) — 이미지 2장 + 설명 (신규)
5. 내지 C (Page 5) — 이미지 1대+2소 + 설명 (기존 유지, onclick 제거)
6. 내지 D (Page 6) — 이미지 4장 그리드 + 설명 (script→정적)
